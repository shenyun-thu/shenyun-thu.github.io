ES常见命令

  • 创建索引: PUT http://192.168.115.81:9200/log?pretty

  • 删除索引: DELETE http://192.168.115.81:9200/twitter?pretty

  • 添加文档:

    POST http://192.168.115.81:9200/log/message/

    {
    ​ “raw”:”this is test5”,
    ​ “hostname”:”sy-test-5”,
    ​ “value”:”10000”,
    ​ “field_add_test”:7

    }

  • 删除文档:

    DELETE http://192.168.115.81:9200/log/message/OykHc2cBah8KRcK6RdIi?pretty

  • 更新文档:

    POST http://192.168.115.81:9200/log/message/mCk6eGcBah8KRcK6KNLS/_update

    {
    ​ “doc”:{
    ​ “raw1”: “this is c raw”
    ​ }
    }

  • 批量添加文档：

    POST http://192.168.115.81:9200/_bulk?pretty

    {“index”:{“_index”:”log”,”_type”:”message”}}\n
    {“name”: “sy” }
    {“index”:{“_index”:”log”,”_type”:”message”}}\n
    {“name”: “wx” }

  • 批量删除文档：(这里有一个比较骚的问题是 bulk方式在批量做的时候需要有换行 是真的需要换 不然会一直报error)

    POST http://192.168.115.81:9200/_bulk?pretty

    {“delete”:{“_id”:”1”}}\n
    {“delete”:{“_id”:”2”}}\n

  • 提前定义映射类型：

    POST http://192.168.115.84:9200/test/_mapping/message

    {
    ​ “message”:{
    ​ “properties”:{
    ​ “timestamp”:{“type”:”date”,

    ​ “format”:”yyyy-MM-dd HH:mm:ss||yyyy-MM-dd||epoch_millis”}
    ​ }
    ​ }
    }

    如果不定义format格式，测试显示

    date默认接受yyyy-MM-ddTHH:mm:ssZ yyyy-MM-dd 正负整数、正负小数、0 别的时间戳格式不行
