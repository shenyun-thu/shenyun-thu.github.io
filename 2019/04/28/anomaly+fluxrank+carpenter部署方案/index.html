anomaly + fluxrank + carpenter 部署测试方案

  • 需要准备的材料

    • 部署包 http://192.168.115.11/files/release/standalone/binary/v0.2.10/
    • carpenter最新版本的包

  • 基本部署步骤

    • 创建aiops用户 本文档默认用户目录为 /home/aiops

    • 将部署包上传至 /home/aiops/media 目录下，解压

    • 到安装目录 cd /home/aiops/media/RELEASE-STANDALONE-BINARY-v0.2.10/script

      • 机器自带python2.x 执行命令 ./install.sh [command] <IP>

        其中 -off 表示安装离线版本 -f 表示含fluxrank -c表示含carpenter -p安装python

        -d指定安装的路径默认为 ~/aiops -id表示导入初始demo

        一般情况下离线POC执行 ./install.sh -off -f -c <IP>

        用最后生成的 Fingerprint 作为校验码获取license校验文件。

        安装完成后执行 source ~/.bash_profile 可通过supervisorctl status查看各组件运行状态

        正常情况下不需要任何操作 如果安装过程中报错或者人工中断安装过程

        cd ~/aiops 执行 ./.uninstall.sh并且删除 rm -r ~/.local 退出用户再重新登录安装即可

      • 机器为安装python2.x 执行命令需要加上 -p参数 并且需要在安装过程需按照下述填写指令

        其余情况与第一种相同

    • 将获得的 license.lic 文件上传至 /home/aiops/aiops/license 目录下

    • 将 carpenter.jar 上传至 /home/aiops/aiops/carpenter 目录下

    • 部署完成
